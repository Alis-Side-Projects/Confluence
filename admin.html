⚙️ Setup Required

To enable visitor tracking, you need a free Supabase project. Takes 5 minutes:

  1. Go to supabase.com → Create a free project
  2. Go to SQL Editor → Run this migration:
CREATE TABLE visitors (
  id bigint GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
  ip text,
  page text,
  browser text,
  os text,
  user_agent text,
  referrer text,
  screen_width int,
  screen_height int,
  language text,
  visited_at timestamptz DEFAULT now()
);

-- Allow anonymous inserts (for tracking)
ALTER TABLE visitors ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow anonymous inserts"
  ON visitors FOR INSERT
  TO anon WITH CHECK (true);
CREATE POLICY "Allow authenticated reads"
  ON visitors FOR SELECT
  TO anon USING (true);
  1. Go to Settings → API → Copy your Project URL and anon public key
  2. Open tracker.js in this repo and paste them into SUPABASE_URL and SUPABASE_KEY
  3. Do the same in admin.html at the bottom of the file
